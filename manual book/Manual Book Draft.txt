# 1. 프로젝트 개요 및 업데이트 히스토리

**"모든 사람, 사물, 공간은 고유한 형태를 갖습니다. 이제 당신만의 스케치를 만들어보세요."**
CAI Sketch to Image는 사용자의 거친 스케치와 텍스트 프롬프트를 바탕으로 고품질의 렌더링 이미지를 실시간으로 생성해주는 AI 기반 웹 어플리케이션입니다.

**🚀 주요 업데이트 연혁:**
- `v1.0 (초기)`: 기본 캔버스 드로잉 및 프롬프트 기반의 단순 이미지 생성 도입
- `v1.5`: 모바일 & 태블릿 반응형 UI 적용, 펜/지우개 굵기 및 동적 텍스트 입력 기능 강화
- `v1.8`: 카메라(환경 렌즈) 연동 및 사진 업로드 기능 추가
- `v2.0 (최종)`: 
  - 관리자(ADMIN) 권한 무제한 토큰 시스템 도입
  - Gemini를 활용한 스케치 메타인지(Metacognitive) 자동 분석 기능 탑재
  - UI 레이어(Z-Index) 최적화 및 로딩 오버레이 시인성 개선

---

# 2. 시작하기 (Getting Started)

서비스를 이용하려면 계정 생성이 필요합니다. 첫 화면의 **팝업(Auth Screen)**에서 진행합니다.

1. **Sign Up (회원가입)**
   - 사용할 이메일과 비밀번호(8자리 이상)를 입력하고 가입합니다.
   - 구글(Google) 또는 카카오(Kakao) 아이디를 연동하여 1초 만에 간편 가입도 가능합니다.
   - *주의: 이미 가입된 이메일로 가입을 시도하면 "중복된 계정입니다" 라는 붉은색 알림이 뜹니다.*
2. **Sign In (로그인)**
   - 계정 생성이 완료되면 로그인하여 접속합니다.
3. **사용 횟수 (Count) 시스템**
   - **일반 유저**: 최초 100회 (이후 과금/충전 필요)
   - **관리자(CRE-TE)**: `∞ 무제한` (시스템에 사전 등록된 이메일만 해당)

---

# 3. 인터페이스 가이드 (주요 화면 설명)

화면은 크게 4가지 영역으로 나뉩니다.

- **🎨 중앙 캔버스 (Canvas)**: 직접 그림을 그리거나, 텍스트를 입력하고, 이미지를 업로드하는 메인 스케치북입니다.
- **🛠 좌측 툴바 (Left Toolbar)**: 
  - 펜(Pen), 지우개(Eraser), 텍스트(Text), 화살표(Arrow), 전체 지우기(Trash) 도구가 있습니다.
  - 최하단에는 라이브러리(Archive)와 설정(Settings) 버튼이 위치합니다.
- **💬 하단 프롬프트 (Bottom Panel)**: 스케치에 대한 추가 설명이나 원하는 스타일을 글로 적는 곳입니다. (예: "미니멀리스트 화이트톤 인테리어")
- **⚙ 우측 제어 패널 (Right Panel)**: 생성하기 전 스타일(선 그리기, 3D 모델 등)을 고르고, 생성 해상도를 선택하며 `GENERATE(생성)` 버튼을 누르는 조종석입니다.

---

# 4. 핵심 기능 사용 방법 (진행 순서)

**STEP 1: 스케치하기**
좌측 툴바를 이용해 캔버스에 아이디어를 스케치합니다.
(우측 상단의 `업로드` 또는 `카메라` 버튼을 눌러 사진을 베이스로 사용할 수도 있습니다.)

**STEP 2: 프롬프트 입력 및 스타일 선택**
하단 입력창에 만들고 싶은 결과물의 묘사를 텍스트로 적습니다.
우측 패널 상단의 `STYLE` 탭을 눌러 원하는 아트 스타일(예: 3D Model, Line art 등)을 선택합니다.

**STEP 3: 생성 (Generate)**
우측 패널 하단의 `GENERATE` 버튼을 누릅니다.
*화면이 어두워지며 AI가 분석을 시작합니다.*

**STEP 4: 결과 확인 및 다운로드**
완성된 이미지가 화면에 나타납니다. 마음에 든다면 `DOWNLOAD` 버튼을 눌러 저장합니다. (고해상도 다운로드 시 일반 유저는 카운트가 추가 차감됩니다.)

---

# 5. 라이브러리 (Library)

내가 과거에 생성했던 멋진 결과물들을 다시 보고 싶다면?

1. **라이브러리 열기**: 좌측 툴바 맨 아래에 있는 **박스 아이콘(Archive)**을 클릭합니다.
2. **기능**:
   - 그리드 형태로 저장된 과거 작업물들이 나열됩니다.
   - 이미지를 클릭하면 해당 작업 과정(스케치 상태)으로 되돌아가 다시 편집할 수 있습니다.
   - `더보기` 글씨를 누르면 당시 사용했던 프롬프트와 AI의 '분석 내용(Diagnosis & Reasoning)'을 확인할 수 있습니다.
   - 휴지통 모양을 누르면 기록을 삭제할 수 있습니다.

# 6. 핵심 구동 프로토콜 (Core Protocols)

CAI 어플리케이션은 뒷단에서 2개의 강력한 인공지능이 릴레이로 동작합니다.

**🧠 1단계: 메타인지 분석 (Gemini AI)**
- 사용자가 스케치를 마치고 프롬프트를 입력하면, 구글의 최신 `Gemini Vision AI`가 사용자의 캔버스 화면을 눈으로 읽습니다.
- 그림의 형태(Geometry), 재질(Materiality), 그리고 사용자의 의도를 추론(`Diagnosis`)하여 숨겨진 의도를 파악합니다.

**✨ 2단계: 고품질 이미지 생성 (Stability AI)**
- 파악된 의도와 텍스트, 그리고 원래의 스케치 이미지를 결합하여 `Stability AI (Stable Diffusion)` 엔진으로 전송합니다.
- ControlNet 기반의 정밀한 선형 인식 기술을 통해, 사용자의 스케치 형태를 훼손하지 않으면서도 실사 수준의 고품질 렌더링으로 변환해냅니다.

# 7. FAQ 및 문제 해결

- **Q. "사용승인 대기중입니다" 또는 "중복된 계정입니다" 라고 뜹니다.**
  - A. 이미 다른 방법(구글/카카오)으로 가입된 이메일입니다. 하단의 'Sign in' 탭으로 이동하여 기존 방식으로 로그인해주세요.
- **Q. 카운트(Count)가 부족하다고 뜹니다.**
  - A. 일반 계정의 무료 생성 횟수를 모두 소진했습니다. VIP 또는 무제한 관리자 권한 부여는 관리팀에 문의하시기 바랍니다.
- **Q. 태블릿/모바일에서 왼쪽 툴바나 오른쪽 패널이 안 보입니다.**
  - A. 기기를 가로(Landscape)로 눕히면 데스크탑과 동일하게 오른쪽 패널 제어 버튼(`</>`)과 추가 도구들이 나타납니다.
- **Q. 그림을 그리지 않으면 생성할 수 없나요?**
  - A. 네, 본 서비스는 '스케치(또는 업로드된 사진)'를 기반으로 동작하므로 캔버스가 비어있으면 `GENERATE` 버튼이 활성화되지 않습니다. 선을 하나라도 그어주세요.
